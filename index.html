How it works

  1. Users send requests to the Load Balancer.
  2. Load Balancer distributes requests to Public and Private cloud servers.
  3. When queued load rises, auto-scaling spins up additional public servers (simulated).
  4. Load reduces and autoscaler scales down to baseline.
